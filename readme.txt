Enables WebPlayer Yahoo! on all pages.

== Description ==

This plugin adds the required javascript for WebPlayer Yahoo!.

For more information visit:

[WebPlayer Yahoo!](http://webplayer.yahoo.com/)

== Installation ==

1. Upload `webplayer-yahoo` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Customize the plugin in the settings page (Admin > Settings > WebPlayer Yahoo!)
